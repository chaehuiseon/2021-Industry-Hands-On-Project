Online Pizza Ordering System

[온라인 피자 주문 시스템]

Requirement Gathering

Admin:
- 피자 추가
- 추가된 피자 조회/수정/삭제
- 들어온 주문 조회/업데이트
- 판매 조회
- 유저 피드백 조회


User:
- 피자 예약
- 로그인
- 마이페이지 방문
- 피자 상세 조회 (장바구니 추가, 바로구매)
- 주문내역 조회
- 피드백 작성
- 비밀번호 변경
